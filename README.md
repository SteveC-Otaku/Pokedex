# 宝可梦图鉴 Pokédex

一个功能完整的宝可梦图鉴应用，支持在线访问和本地使用。

## ✨ 功能特性

- 📚 **完整的宝可梦数据**：包含所有世代的宝可梦信息
- 🔍 **强大的搜索功能**：支持按名称、编号搜索
- 🎯 **多维度筛选**：
  - 按世代/地区筛选
  - 按属性筛选（支持多选）
  - 按种族值排序
- 📊 **详细信息展示**：
  - 基础信息（属性、身高、体重等）
  - 种族值可视化
  - 属性相克关系
  - 招式列表（包含升级、技能机、教学、蛋招式）
  - 出现地点（按世代分组）
  - 进化链
- 🎲 **捕获率计算器**：
  - 支持多种精灵球
  - 支持异常状态加成
  - 实时计算捕获概率
- ❤️ **收藏功能**：收藏喜欢的宝可梦
- ⚖️ **对比功能**：对比两个宝可梦的属性、种族值等
- 👥 **队伍构建器**：
  - 构建最多 6 只宝可梦的队伍
  - 选择形态、特性、努力值、招式
  - 保存多个队伍配置
  - 查看队伍类型覆盖分析
- 🔄 **形态切换**：支持查看和切换宝可梦的不同形态（如洛托姆的各种形态）
- 🌐 **多语言支持**：中文/英文/日文切换
- 🎨 **主题切换**：支持亮色/暗色主题

## 💻 本地使用（可选）

如果你想在本地运行这个应用：

### 环境要求

- Node.js 18+ 
- pnpm (推荐) 或 npm

### 安装和运行

```bash
# 安装依赖
pnpm install

# 开发模式运行
pnpm dev
```

访问 [http://localhost:3000](http://localhost:3000)

### 构建静态版本

```bash
# 构建生产版本
pnpm build
```

构建后的文件在 `out/` 目录，可以直接部署到任何静态网站托管服务。

## 📝 使用说明

### 搜索宝可梦
- 在顶部搜索框输入宝可梦名称或编号
- 支持中文、英文、日文搜索

### 筛选和排序
- **世代筛选**：选择要查看的世代
- **属性筛选**：选择一个或多个属性，会显示同时拥有所有选中属性的宝可梦
- **排序**：按编号、名称或种族值排序

### 查看详细信息
- 点击宝可梦卡片查看详细信息
- 切换不同标签页查看：种族值、特性、进化、属性克制、招式、出现地点

### 形态切换
- 对于有多个形态的宝可梦（如洛托姆），可以在详情页切换形态
- 不同形态可能有不同的属性、种族值和特性

### 收藏功能
- 点击宝可梦卡片上的心形图标收藏
- 在"收藏"标签页查看所有收藏的宝可梦

### 对比功能
- 在"对比"标签页选择两个宝可梦进行对比
- 可以查看属性、种族值、类型克制等差异

### 队伍构建器
- 在"队伍构建器"标签页构建你的队伍
- 为每个位置选择宝可梦、形态、特性、努力值和招式
- 点击"保存队伍"保存配置，可以保存多个队伍
- 查看队伍的类型覆盖分析

### 捕获率计算器
- 在宝可梦详情页底部有捕获率计算器
- 输入等级、当前 HP、选择异常状态和精灵球类型
- 实时查看捕获概率
